# InternMatch

Internship recommendation MVP for IIT Patna Semester 2 capstone.  
This app recommends relevant internships based on user-entered skills and current market openings.

## Live Demo
🔗 [Streamlit Deployment](https://internmatchiitp.streamlit.app/)

## Features
- Loads cleaned internship data
- Accepts user skills/input
- Recommends relevant internships
- Shows market insights in Streamlit

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
