# InternMatch

Internship recommendation MVP for IIT Patna Semester 2 capstone.  

This app recommends relevant internships based on user-entered skills and current market openings.

## Live Demo
[Open the Streamlit app](https://internmatchiitp.streamlit.app/)

## Features
- Loads cleaned internship data
- Accepts skill-based search input
- Recommends matching internships
- Shows simple dataset insights

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
