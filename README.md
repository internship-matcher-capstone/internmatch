# InternMatch

A Streamlit-based internship matching project built for the IIT Patna Semester 2 capstone.
The app takes user-entered skills, compares them with internship listings, and shows the most relevant matches along with some basic dataset insights.

## Live Demo
[Open the Streamlit app](https://internmatchiitp.streamlit.app/)

## Features
- Loads cleaned internship data
- Accepts skill-based search input
- Recommends matching internships
- Shows simple dataset insights

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
